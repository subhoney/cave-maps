This is the React frontend for the Yucatan Cenote Cave Maps project. It displays cenote information, cave map images, and a bio for Roberto Castillo, styled in Dive Rite colors.

## Features
- List and detail view of cenotes
- Cave map images (served from `/public/img/`)
- Roberto Castillo bio section
- Responsive, Dive Rite-themed design

## Setup
1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm start
   ```
3. Access the app at [http://localhost:3000](http://localhost:3000)

## Static Assets
- Place cave map images and bio pictures in `public/img/` for use in the app.

## Customization
- Update colors and layout in `src/diveRiteTheme.css`
- Edit diver bio and images in `src/App.js`
